How to run the CREDIT CARD Application Management System (CCAMS) Project

1. Download the  zip file

2. Extract the file and copy ccams folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name ccamsdb

6. Import ccamsdb.sql file(given inside the zip package in SQL file folder)

7.Run the script http://localhost/ccams (frontend)
